# Shopping Store Exercise

An exercise site to practice _Angular 8_.

### The site use:

- Reactive Forms.
- NgRx.
- Basic Lazy Loading.
- Loading data and authentication using Firebase.
- Material Angular.
- Directives.
- Pipes.
- Observers.
- A few tests.
